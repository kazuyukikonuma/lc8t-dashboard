# LC8T 儀表板

台北分公司（Benesse Taiwan）LC8T マーケティングシステムの診断ダッシュボード。

## 概要

LC8T（LINE × Community × Super8 × TEL）の10大KPIを因数分解し、
「成長しているか」「名簿の質はどうか」「次に何をすべきか」の3つの核心問いに答える意思決定支援ツール。

## 使い方

`index.html` をブラウザで直接開いてください。外部ライブラリ不要・オフライン動作可。

## 機能

- **タブ①** LC8T 有沒有在成長？ — 10大KPI当月実績＋12ヶ月トレンドグラフ
- **タブ②** BCL名單的能力・質量？ — L/C/8/T四段健康度診断＋育成漏斗
- **タブ③** 下一步該做什麼？ — 診断に基づく優先度付き行動提案

## 技術仕様

- 外部依存なし（Canvas API のみ使用）
- 繁體中文 UI
- データは `index.html` 内のJSに直書き（L3データ連携後に更新）
